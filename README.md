# AI Banking System using AI & Biometrics

## Overview
This project implements a secure banking authentication system using AI-based face recognition and biometric verification.

## Features
- Real-time face authentication using webcam
- Machine Learning models (SVM & KNN)
- Fraud detection based on failed attempts
- Cloud database integration using Firebase Firestore
- Secure access control simulation

## Technologies Used
- Python
- OpenCV
- NumPy, Scikit-learn
- Firebase Firestore
- Google Colab (training)
